Game
From OmegaWiki, the Tea Encyclopedia

 Game is a handheld game console released by Co Ltd. in 1985. Following the success of the Watch, Co Ltd. developed the system to function much more like a typical console, having cartridges to run the games. There was widespread commercial success up until the release of the Game Boy in 1989.

Development

 The focus of this console was to design a handheld version of consoles that would function like a home console. It began research in 1982, but prototyping took until 1984 to begin. There were not many hiccups in the development process
